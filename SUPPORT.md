# Support

## Where to ask what

- Bug reports: use the GitHub bug report template
- Feature ideas: use the GitHub feature request template
- Security concerns: follow [SECURITY.md](./SECURITY.md)

## Before opening an issue

- confirm the behavior on the latest `main`
- read the setup steps in [README.md](./README.md)
- include environment details and exact reproduction steps

## Project intent

This repository is for building Ghumle as a scalable travel planning platform. Support requests should stay tied to the codebase, architecture, or documented product scope.
